# All Things Onion

## Challenge Summary
All Things Onion is a beginner-friendly steganography challenge.  
Players are given a single image file and must identify and extract hidden information embedded within it.

Prompt - 
A certain chef was obsessed with onion. She had a perfect recipe for onion soup, refined through repetition and patience. But what set this recipe apart was her choice of ingredients and the side dish that accompanied it. 
  She finished the soup with a thin strip of orange zest, rolled neatly and fixed to the rim of the bowl, measuring no longer than 48 millimeters.
  Alongside it, she prepared yam, turning and shaping it through 51 deliberate movements before it was ready.
  At the very end, she added yolk, watching closely and counting 51 slow breaths before the dish was complete.
  Everything had its moment, but the onion always came first.

---

## Local Build / Run Instructions
This challenge does **not** require a local build or runtime service.

There is no Docker container, server, or executable associated with this challenge.  
Players interact solely with the provided image file.

## Files
- `not_onion.jpg` — Image containing hidden data required to solve the challenge.
